# ☕ КавоБізнес — Система обліку

Telegram бот + Веб-дашборд для обліку спільного кавового бізнесу.

---

## 📁 Структура проекту

```
kavobiz/
├── bot.py              — Telegram бот (AI, голос, кнопки)
├── dashboard.html      — Веб-дашборд (відкрити в браузері)
├── requirements.txt    — Python залежності
├── .env.example        — Шаблон змінних середовища
├── Dockerfile          — Docker образ
├── docker-compose.yml  — Docker Compose
├── Procfile            — Railway/Heroku
└── railway.json        — Конфіг Railway
```

---

## 🚀 ВАРІАНТ 1 — Railway (найпростіше, безкоштовно)

### Крок 1 — Підготовка
1. Зареєструйся на [github.com](https://github.com)
2. Створи новий репозиторій: **New repository → kavobiz → Create**
3. Завантаж всі файли проекту в репозиторій

### Крок 2 — Деплой на Railway
1. Зайди на [railway.app](https://railway.app)
2. **New Project → Deploy from GitHub repo → обери kavobiz**
3. Railway автоматично знайде `Procfile` і запустить бота

### Крок 3 — Змінні середовища
В Railway → твій проект → **Variables → Add Variable**:
```
BOT_TOKEN      = токен від @BotFather
GROQ_API_KEY   = ключ з console.groq.com
ID_VOLODYMYR   = твій Telegram ID
ID_VYGRAN      = ID Вигрна
```

### Крок 4 — Готово!
Бот запущений 24/7. Безкоштовно 500 год/місяць.

## 🌐 Веб-дашборд

Файл `dashboard.html` — відкрий **просто в браузері**, нічого встановлювати не треба.

**На телефоні:**
- Chrome: відкрий файл → три крапки → **"Додати на головний екран"**
- Safari (iPhone): відкрий → поділитись → **"На екран «Додому»"**

Буде виглядати як додаток.

**Синхронізація бот ↔ веб:**
- Бот і веб використовують окремі бази (бот — файл на сервері, веб — localStorage браузера)
- Для синхронізації: в боті `/export` → отримуєш JSON → в веб-дашборді ⚙️ → **Імпорт JSON**

## 🔑 Де отримати токени

### Telegram Bot Token (безкоштовно):
1. Відкрий [@BotFather](https://t.me/BotFather) в Telegram
2. Напиши `/newbot`
3. Дай назву боту, наприклад: `КавоБізнес`
4. Дай username, наприклад: `kavobiz_bot`
5. Скопіюй токен

### Groq API Key (безкоштовно, для AI і голосу):
1. Зайди на [console.groq.com](https://console.groq.com)
2. Sign up (можна через Google)
3. **API Keys → Create API Key**
4. Скопіюй ключ

### Твій Telegram ID:
1. Напиши [@userinfobot](https://t.me/userinfobot) в Telegram
2. Він одразу відповість твій ID
3. Попроси Вигрна зробити те саме

## ☕ Логіка розрахунків

| Товар | Мої точки | Точки Вигрна |
|-------|-----------|--------------|
| Кава (продаж 700) | Вигрну: 530 грн | Вигрну: 630 грн |
| Комплект (продаж 850) | Вигрну: 680 грн | Вигрну: 780 грн |
| Молоко / Айріш | Вигрну: 0 грн | Вигрну: 100 грн |

**Баланс:** `> 0` = Вигран винен Володимиру / `< 0` = Володимир винен Вигрну

Ціни можна змінити в **Налаштуваннях** веб-дашборду або при кожному продажу окремо.

## 📱 Команди бота

| Команда | Дія |
|---------|-----|
| `/start` | Головне меню |
| `/balance` | Баланс і залишки |
| `/report` | Місячний звіт |
| `/export` | Excel файл |
| `/history` | Останні 10 операцій |
| `/stock` | Залишки на складі |
| `/points` | Список точок |
| `/addpoint` | Додати точку |
| `/setpin` | PIN для балансу |
| `/undo` | Скасувати останнє |

**Вільний текст:**
- `Оляниця 5 кав` → продаж
- `+20 комплектів` → постачання
- `Передав 5000` → виплата
- Голосові повідомлення → транскрибація і запис
